# Telecom-Churn-Analysis
This is a Telecom Customer Churn Analysis dashboard. The dataset contains customer information such as gender, senior citizen status, tenure, internet service type, contract type, payment method, monthly charges, total charges, and whether the customer has churned (left the company) or not.

Categorical Analytics Project (Power BI)

designed a Power BI dashboard tailored for churn analysis for a telecom company, with a strong emphasis on categorical charts to provide intuitive and impactful visualizations for a comprehensive understanding of the data.
